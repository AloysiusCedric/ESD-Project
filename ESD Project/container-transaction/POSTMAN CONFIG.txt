[POST METHOD]
Paying for stay
http://localhost:5001/transaction

BookingNum will be autogenerated

Sample Data to put in POSTMAN(will be received from booking microservice):
{
    "houseId": 1,
    "transactionId":"",
    "startDate": "2023-04-01",
    "endDate": "2023-04-07",
    "status": "confirmed",
    "paymentId":"2GC1726685321561L"
}


result:

{
    "Message": "New entry created successfully!",
    "bookingNum": "K1UPQ5"
}



[POST METHOD]
Cancel booking number for a specific bookingNum (edit the postman bookingNum)
http://localhost:5001/transaction/347F50/cancel

result:

{
    "message": "Transaction with bookingNum 347F50 has been cancelled.",
    "paymentId": "2GC1726685321561L"
}
